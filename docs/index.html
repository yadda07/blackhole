Functions

Two functions, all the power

recover_json()

Automatic capture of each operation. OLD values are serialized to JSONB with timestamp and user.

recover()

Historical data reconstruction. Filtering by table, operation, period and user.

Trigger behavior

OperationCaptureSemantics
DELETEOLDLast trace before deletion
UPDATEOLDPrevious state preserved
INSERTNEWBirth recorded

Recovery example

SELECT rip_avg_nge.recover(
    'rip_avg_nge',           -- schema
    'infra_pt_pot',          -- table
    'DELETE',                -- operation
    '2025-01-01',            -- debut
    '2025-01-31',            -- fin
    100                      -- limite
);

-- Donnees restaurees dans :
SELECT * FROM temp_rip_avg_nge_infra_pt_pot;
